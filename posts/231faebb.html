基础知识

变量

变量的命名规则

变量不能随便写，其有一套命名规则:
1.变量名由数字、大小写字母与下划线(“_”)组成
2.变量名不能以数字开头，同时也不能单个数字作为变量名，如：1
3.变量名不能包含“#”等特殊字符

注意：只要有双引号或者单引号的数据，如”True”,”123”等，这些是字符串，不要被骗到了，整数型和布尔型等其他类型是不带单双引号的

布尔型、逻辑型 bool True或False 看到True或False则为布尔型
单浮点型 float 1.0 小数点后面有一位的数字则为单浮点型

运算符

算数运算符

算术运算符 作用 例子
+ 两个对象相加 a + b
- 两个对象相减 a - b
* 两个对象相乘 a * b
/ 两个对象相除 a / b
% 取除法的余数 a % b
// 取整除 a // b
** 取其n此幂 a ** b

比较运算符

比较运算符 作用 例子
== 判断两个对象是否相等 a == b
!= 判断两个对象是否不相等 a != b
> 判断一个对象是否大于另一对象 a > b
< 判断一个对象是否小于另一对象 a <> b
>= 判断一个对象是否大于等于另一对象 a >= b
<= 判断一个对象是否小于等于另一对象 a <= b
结果为布尔型、逻辑型，也就是False或True

逻辑运算符

此节部分内容摘自https://www.runoob.com/python3/python3-basic-operators.html#ysf1

逻辑运算符 逻辑表达式 描述
and x and y x与y为布尔型，当x与y都为True时，此表达式整体为True，反之有一个为False，整体为False
or x or y x与y为布尔型，当x与y其中有一个为True时，此表达式整体为True，都为False时，整体为False
not not x x为布尔型，如果x为True，则not x整体返回False，简单来说，就是将x的值变为与其相反的布尔型

逻辑运算符的优先级：
秘诀：先not后and在or
这句话什么意思呢？
逻辑表达式中，先将not单体的值换出来，然后再去看and那一整体，最后再去看or的整体

条件控制

Python 条件语句是通过一条或多条语句的执行结果（True 或者 False）来决定执行的代码块。（https://www.runoob.com/python3/python3-conditional-statements.html ）

if-condition

条件是布尔型
如果 “条件1” 为 True 将执行 “语句块1” 块语句
如果 “条件1” 为False，将判断 “条件2”
如果 “条件2” 为 True 将执行 “语句块2” 块语句
如果 “条件2” 为False，将判断”条件3”
如果 “条件3” 为 True 将执行 “语句块3”
如果 “挑拣3” 为False，将不执行任何语句

每个条件后面要使用冒号;使用缩进来划分语句块,相同缩进数的语句在一起组成一个语句块.
